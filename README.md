# Link in Bio Website

A simple "link in bio" page built with HTML and CSS.

---

## Build Time

This project was created in approximately 15 minutes as a quick test to see how fast I can make a simple static website.

---

## Features

- Profile section with avatar  
- Name and short description  
- Social media links  
- Clean responsive layout  
- Hover effects on links  

---

## Tech Stack

- HTML5  
- CSS3  

---

## Deployment

Deployed using GitHub Pages.

Steps:
1. Upload files to a GitHub repository
2. Go to Settings → Pages
3. Select branch: main
4. Save and wait for deployment

---

## Purpose

This project was made as a quick practice build to test speed, layout design, and deployment workflow for static websites.
